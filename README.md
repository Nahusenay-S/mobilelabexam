 Daily Note and Quote App

A simple and user-friendly application that helps users add, edit, and delete daily notes and motivational quotes efficiently. The app is designed to improve productivity by allowing users to organize their thoughts and keep important notes in one place.

Features

* Add new notes and view  quotes
* Edit existing notes
* Delete unwanted notes
* Simple and clean user interface
* Easy note management
